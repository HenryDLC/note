# 装饰器
> 本质:装饰器就是一个闭包
### 定义及使用装饰器
```python
import time

# 定义装饰器
def time_log(fun):
    def inner():
        print(time.time())
        fun()
        print(time.time())

    return inner

# 使用装饰器
@time_log
def get_name():
    print('Tom')


get_name()
```
#### 执行过程:
get_name = time_log(get_name) => get_name = inner
1. 调用get_name
get_name()相当于 直接调用 inner()
