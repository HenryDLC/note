# 匿名函数
> 生成数,可用list()方法生成列表
python3 返回可迭代对象
python2 返回列表(python2中xrange()函数返回可迭代对象)
### range函数
```python
# range(起始值, 截止值-1, 步长)  返回:可迭代对象
a = range(1, 10, 2)
for i in a:
    print(i)
# >>> 1
# >>> 3
# >>> 5
# >>> 7
# >>> 9
```
### lambda匿名函数
> lambda 传入参数 : 返回值
```python
# lambda 传入参数 : 返回值

foo = lambda num1, num2: num1 + num2
print(foo(2, 3)) # >>> 5
```

### map函数
> 对列表进行操作
map(函数方法, 数据)
返回可迭代对象
```python
# 对列表进行操作
# map(函数方法, 数据)
# 返回可迭代对象
def foo(x):
    return x ** x


ret = map(foo, [0, 2, 4, 6, 8])
print(list(ret))  # >>> [1, 4, 256, 46656, 16777216]
# 也可写成
print(list(map(lambda x: x ** x, list(range(0, 10, 2)))))  # >>> [1, 4, 256, 46656, 16777216]
```
