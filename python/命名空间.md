# 命名空间
导入包时,如果本地的变量和导入包中的变量重名,那么我们调用谁就引用谁的变量,不会被覆盖
```python
spacename = 'test'


def print_spacename():
    print(spacename)
```
```python
import test

spacename = 'main'
print(spacename)  # >>> main

test.print_spacename()  # >>> test
```
