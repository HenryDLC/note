# 进程
> 通过os模块的fork()方法可以传建一个子进程,并且将子进程的PID返回给父进程的python解释器,将返回值`0`返回给子进程
> 
```python
import os, time

# 创建一个进程(子进程)
ret = os.fork()

if ret == 0:
    for i in range(5):
        print('子进程')
        time.sleep(0.1)

else:
    for i in range(5):
        print("父进程,子进程的PID为{}".format(ret))
        time.sleep(0.1)

# >>> 父进程,子进程的PID为29569
# >>> 子进程
# >>> 子进程
# >>> 父进程,子进程的PID为29569
# >>> 子进程
# >>> 父进程,子进程的PID为29569
# >>> 子进程
# >>> 父进程,子进程的PID为29569
# >>> 子进程
# >>> 父进程,子进程的PID为29569
```
