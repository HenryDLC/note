# 进程
> 创建:通过os模块的fork()方法可以传建一个子进程,并且将子进程的PID返回给父进程的python解释器,将返回值`0`返回给子进程

> 资源:创建子进程的同时,会将所有的资源和当前的运行状态完全复制一份给子进程

> 为了演示下过,我们需要将两个代码块都做sleep(0.1)的处理,每次执行后都可让程序过渡到下个时间片上,如果不对

```python
import os, time

# 创建一个进程(子进程)
ret = os.fork()

if ret == 0:
    for i in range(5):
        print('子进程')
        time.sleep(0.1)

else:
    for i in range(5):
        print("父进程,子进程的PID为{}".format(ret))
        time.sleep(0.1)

# >>> 父进程,子进程的PID为29569
# >>> 子进程
# >>> 子进程
# >>> 父进程,子进程的PID为29569
# >>> 子进程
# >>> 父进程,子进程的PID为29569
# >>> 子进程
# >>> 父进程,子进程的PID为29569
# >>> 子进程
# >>> 父进程,子进程的PID为29569
```
