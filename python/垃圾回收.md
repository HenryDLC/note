# 垃圾回收
### 引用计数
> 当一个对象的内存空间被一个'容器'所引用,比如`a = 1`整数对象'1'被a所引用,那么1的引用就会被+1,如果`b = a`a的对象被b引用,就再做+1操作,但当我们连续`del`两个引用,那么1的整形对象就会被-2处理,当引用计数归零,python解释器就被将整数1的内存地址删除,节省资源.
```python

a = 1
b = a
print(b) # >>> 1
del b
del a
print(b) # >>> NameError: name 'b' is not defined
```
